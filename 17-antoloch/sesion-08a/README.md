#  Apuntes KiCad

**En esta clase instalamos KiCad, no fue difícil de instalar.**  
**Realizamos nuestro primer proyecto en KiCad.**

##  Flujo de trabajo:)))

1. Dibujar esquemático en **kicad_sch**
2. Asociar huellas (físico) a símbolos (abstracciones/ideas)
3. Abrir PCB para interpretar esquemático
4. Definir tamaño pistas
5. Repartir componentes
6. Rutear
7. Adornar
